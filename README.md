# Materiais-Estudo
Repositório destinado aos materiais de estudo desenvolvidos para as disciplinas de desenvolvimento de software e programação.

## Tecnologias
MkDocs - ferramenta que transforma arquivos markdown em um site completo
Instalação:
```
pip install mkdocs
```
Criação de Projeto:
```
mkdocs new my-project
cd my-project
```
Iniciar o Servidor:
```
mkdocs serv
```
## Tema:
pip install mkdocs-material
