# Página de Perfil

Projeto introdutório de Front-End desenvolvido com **HTML, CSS e JavaScript**, com o objetivo de ensinar conceitos fundamentais de desenvolvimento web.

Durante a construção da aplicação, os alunos aprendem sobre:

* Estruturação de páginas utilizando HTML;
* Estilização e responsividade com CSS;
* Manipulação do DOM com JavaScript;
* Organização de arquivos e boas práticas de projeto.

## Objetivo

Desenvolver uma página de perfil digital que apresente informações pessoais, imagem de perfil, links e outros elementos visuais, permitindo a aplicação prática dos conceitos estudados.

## Resultados de Aprendizagem

Ao concluir este projeto, o aluno será capaz de:

* Criar páginas web estruturadas com HTML;

* Manipular elementos da página através do JavaScript;
