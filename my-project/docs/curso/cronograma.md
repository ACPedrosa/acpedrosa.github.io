# Programação de Aplicativos

## Sobre a Unidade Curricular

Esta Unidade Curricular tem como objetivo desenvolver capacidades técnicas relacionadas à criação de aplicações computacionais, utilizando linguagem de programação, boas práticas, integração com banco de dados e testes, respeitando padrões de qualidade, segurança e organização de código.

---

## Objetivo Geral

Propiciar o desenvolvimento de aplicações por meio da programação, considerando:

- Qualidade de código
- Organização e legibilidade
- Reutilização
- Testabilidade
- Integração com banco de dados
- Boas práticas de desenvolvimento

---

## Competências Desenvolvidas

Ao longo desta UC, o aluno será capaz de:

- Preparar e configurar ambiente de desenvolvimento
- Aplicar JavaScript para construção de aplicações
- Manipular DOM e criar interatividade
- Organizar projetos com padrões profissionais
- Utilizar Git para versionamento e rastreabilidade

---

## Estrutura do Repositório

Este repositório está organizado por **módulos de aprendizagem**, representando a evolução do desenvolvimento das aplicações.

```
prog-aplicativos/
│
├── modulo-01-frontend/
├── modulo-02-javascript/
├── modulo-03-dom/
└── modulo-04-projeto-final/
```
## Módulos da UC

### Módulo 01 – Frontend (HTML e CSS)
Revisão prática de HTML e CSS com foco em criação de layouts.

Conteúdos:
- Estrutura HTML
- Flexbox
- Grid
- Layouts responsivos

### Módulo 02 – JavaScript (Lógica de Programação)
Desenvolvimento de lógica computacional utilizando JavaScript.

Conteúdos:
- Variáveis
- Condicionais
- Laços de repetição
- Funções

---

### Módulo 03 – DOM e Interatividade
Integração entre JavaScript e HTML.

Conteúdos:
- Eventos
- Manipulação do DOM
- Formulários
- Atualização dinâmica de interface

---

### Módulo 04 – Projeto Final
Desenvolvimento de uma aplicação completa integrando todos os conhecimentos adquiridos.
